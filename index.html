Smartquizbeta

New quiz for vocational student

Download this project as a .zip file Download this project as a tar.gz file

http://elvicky.github.com/smartquizbeta

Kuis ini digunakan untuk menguji kemampuan siswa, khususnya untuk siswa SMK dalam bidang TIK. Soal-soal yang diujikan adalah soal mengenai pengetahuan mendasar tentang beberapa software basic pada computer atau laptop, misal microsoft word, microsoft excel, dan lain lain.

Kuis ini memberikan feedback langsung di tiap-tiap soal, sehingga seorang siswa bisa langsung mengetahui jawaban yang benar dan jawaban yang salah. Dalam feedback juga disertakan sedikit informasi tambahan mengenai jawaban yang benar. Di akhir, kami juga memberikan feedback berupa berapa soal yang terjawab benar, kemudian diberikan juga ranking bagi siswa. sekaligus saran yang mungkin bisa membantu siswa meningkatkan kemampuan belajarnya.

Susunan kuis ini menggunakan teknologi berbasis javascript. Dengan beberapa bantuan plugin yaitu JQuery, SlickQuiz, JSON, dan lain-lain. Khusus dalam pengerjaan animasi, kami menggunakan Sprite Animation. Dan untuk kuisnya kami menggunakan plugin SlickQuiz.

Untuk demo dari quiz ini, silahkan akses di alamat http://elvicky.github.com/smartquizbeta
